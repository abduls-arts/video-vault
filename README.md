# video-vault
# 🎬 VideoVault – Universal Video Downloader

A powerful Python-based tool to extract and download videos from websites with preview support.

## 🚀 Features
- 🔍 Fetch video info from URL
- 🎥 Preview before download
- 📥 Download in best quality
- 🔄 Convert to MP4
- 📝 Embed subtitles (if available)

## 🛠️ Tech Stack
- Python
- yt-dlp / requests
- Tkinter (GUI)


## ⚙️ Installation

```bash
git clone https://github.com/abduls-arts/video-vault.git
# Install dependencies
pip install yt-dlp pillow

# Optional: install ffmpeg for MP4 conversion & audio extraction
# Ubuntu/Debian:
sudo apt install ffmpeg
# Windows: https://ffmpeg.org/download.html

# Run the app
python video_vault_v1.py
